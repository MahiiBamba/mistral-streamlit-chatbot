# Mistral Streamlit Chatbot

A simple AI chatbot built using:
- Streamlit
- Mistral AI API

## Features
- Chat interface
- Secure API key using secrets.toml
- Session-based chat history

## Setup

1. Create virtual environment
2. Install dependencies:
   pip install streamlit mistralai
3. Add API key inside .streamlit/secrets.toml
4. Run:
   streamlit run app.py
